# Коллаб: ты + друг + я (код)

## Перед / после работы (агент — навсегда)

1. **`git pull`** — друг мог запушить; сверить версии.  
2. Прочитать **`docs/MASTER_PLAN.md`**.  
3. Правки.  
4. Сам **`commit` + `push`** (remote всегда актуален).  
5. **Не** коммитить/пушить — только если явно сказали «не коммить / не пушь».  

Правило для AI: корневой **`AGENTS.md`**.

## Зачем так

| Слой | Кто владеет | Где живёт |
|------|-------------|-----------|
| **Код** (системы, баланс) | Я / ты через Git | `src/` в репозитории |
| **Визуал** (terrain, модели, свет) | Ты + друг | **Roblox Place** (Studio) |
| **Связка** | Договорённость | Папки `Loc0N.Art` + Git для кода |

Rojo **не трогает** `Workspace` (см. `default.project.json`) — ваши постройки **не затираются** при синхроне скриптов.

Код **никогда не удаляет** `Loc0N.Art`.

## 2. Структура в Studio (святое)

```
Workspace.World
├── _README
├── Lobby/
│   ├── Art/          ← ваш лобби-арт
│   └── Scaffold/     ← серые гайды (можно удалить)
└── Locations/
    ├── Loc01/
    │   ├── Art/              ⭐ СЮДА ВСЁ ВАШЕ
    │   │   ├── Terrain notes / models
    │   │   ├── Trees, rocks…
    │   │   └── PlayerSpawn   (переместите сюда свой спавн)
    │   └── Scaffold/         ← гайды зон (можно скрыть/удалить)
    ├── Loc02/ …
    ├── Loc03/ …
    └── Loc04/ …
```

**Правила:**
1. Всё красивое → только в **`Art`**.
2. Не переименовывать `Loc01`…`Loc04`.
3. Спавн: часть с именем **`PlayerSpawn`** (где угодно внутри Loc0N).
4. `Scaffold` — временный; когда floor готов — можно удалить `Scaffold.FloorGuide`.

## 4. Версии (чтобы не потерять работу)

```

```

Или в CHANGELOG нумеруйте: `Loc01 art v3`.

## 5. Как проверить что «всё работает» с картой

| Проверка | Ожидание |
|----------|----------|
| Play Solo | 4 территории, телепорт на Loc01 |
| Переход Loc2 (если unlock) | TP на спавн Loc02 |
| Клик/автокликер | Урон, монеты |
| Поставить дерево в `Loc01.Art` | После Stop+Play дерево **на месте** |
| Rojo sync скрипта | Дерево **не исчезло** |
| Удалить Scaffold | Игра всё ещё TP на PlayerSpawn в Art |

Если что-то пропало — вы положили арт **не в Art**, а в Scaffold и включили `REBUILD_SCAFFOLD` (оставьте **false**).
