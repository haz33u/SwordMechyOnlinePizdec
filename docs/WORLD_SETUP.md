# Мир Phase 1 — 4 крупные территории

См. также: **[COLLAB.md](./COLLAB.md)** (работа с другом и версиями).

## Масштаб

| | |
|--|--|
| **1 локация** | **1200 × 1200** studs |
| Зазор | 400 studs |
| Сетка | **2 × 2** |
| Footprint | ~**3200 × 3200** studs |
| Переход пешком через локацию | ~**75 сек** |

Раньше было 280 — мало для атмосферы. Сейчас в ~4× больше по стороне (~16× по площади).

## 4 территории

```
        X=0                X=1600
Z=0     Loc01              Loc02
        Тёмный лес         Пиратский берег

Z=1600  Loc03              Loc04
        Земли шиноби       Полярная тундра
```

| ID | Имя | unlockPower (скелет) |
|----|-----|----------------------|
| 1 | Тёмный лес | 0 |
| 2 | Пиратский берег | 2_000 |
| 3 | Земли шиноби | 15_000 |
| 4 | Полярная тундра | 80_000 |

## Папки (арт безопасен)

```
Loc0N/
  Art/        ← ВЫ и друг. Код НЕ трогает.
  Scaffold/   ← жёлтые углы, зоны, серый пол-гайд
```

## Как начать в Studio

1. New Place  
2. Rojo connect (код)  
3. Play → `Workspace.World` с 4 локациями  
4. Stop → Publish / Save  
5. Строите **только в `Art`**  
6. Team Create → друг  

Подробности коллаба → `docs/COLLAB.md`.
