# Основные игровые системы (скелет)

> Без доната. Одна валюта: **Coins**.  
> **Клики** — главная модель заработка.

---

## 0. Главный loop (заработок)

```
КЛИК (ручной или авто)
   ↓
Удар по мобу (если прошёл CD)
   ↓
+1 Click  |  +Damage  |  (crit?)
   ↓
Моб умер?
   ├─ нет  → продолжаем кликать
   └─ да   → +Power +Coins +шанс Меч/лут
                ↓
         Сильнее клик → быстрее киллы → больше монет
                ↓
         Апы / мечи / петы / ауры / rebirth
                ↓
         Ещё выше CPS и урон за клик
```

**Метрики игрока:**
| Метрика | Смысл |
|---------|--------|
| **CPS** | Кликов в секунду (скорость атаки) |
| **Урон/клик** | = TotalPower (×2 если крит) |
| **DPS** | CPS × урон/клик |
| **totalClicks** | Всего кликов за сезон/акк |
| **lifetimeDamage** | Суммарный урон → Rebirth |
| **lifetimePower** | Сила с мобов |
| **coins** | Soft-валюта |

## 1. Клики и автокликер ⭐ CORE

### Ручной клик
- LMB / кнопка «Удар» / тап
- Сервер принимает `Swing` не чаще, чем `1/CPS`
- Успех = 1 click + урон

### Автокликер
- Тоггл в HUD
- Клиент шлёт Swing с интервалом `1/CPS`
- Сервер **тот же rate-limit** (нельзя читерить сверх CPS)
- Урон = как у ручного (`AUTO_DAMAGE_MULT = 1.0`)
- В скелете: **открыт сразу** (F2P). Позже: квест/rebirth

### От чего растёт CPS
1. Ап **Скорость удара** (ClickSpeed)
2. Чары меча **Скорость удара**
3. % скорости с **питомцев**
4. (позже) второй меч под speed, ивенты

```
CPS = clamp(1 / SwingCooldown, MIN_CPS, MAX_CPS)
SwingCooldown = BaseCD / (1 + attackSpeed% / 100)
```

## 2. Сила и урон

```
TotalPower =
  (Base + lifetimePower)
  × RebirthMult
  × WeaponMult          -- main + 0.5×offhand
  × (1 + power% / 100)  -- чары, петы, ауры, реликвии, ап Сила
  × (1 + damage% / 100)
```

Урон за клик ≈ TotalPower (крит ×2).

---

## 3. Перерождение (Rebirth)

- Условие: `lifetimeDamage >= Cost(n)`
- Soft: **не** сбрасывает мечи/петов/локи
- Даёт `RebirthMult *= (1 + bonus)`
- Ускоряет весь дальнейший фарм

---

## 4. Апы персонажа (NPC / меню)

| Ап | Эффект |
|----|--------|
| Скорость бега | WalkSpeed |
| Рюкзак | слоты инвентаря (UI) |
| Сила | +% power |
| **Скорость удара** | **+CPS** |
| Крит | шанс ×2 урона |
| Удача | +% дроп |

Платятся **Coins**.

---

## 5. Мечи

- Дроп с мобов
- Main + Offhand (50% mult)
- Билд: урон-меч + speed-меч
- Продажа, ban-list мусора
- Зачарование-рулетка (%)

---

## 6. Питомцы

- Кейс → rarity
- Слоты 1→7 (квесты/данжи)
- Команда даёт % силы / монет / speed
- Feed за coins

---

## 7. Ауры

- 1 активная
- % силы / урона / монет
- Кейс за coins (скелет)

---

## 8. Реликвии / бижутерия

- Из данжей
- Большие % силы/урона
- Слоты экипа

---

## 9. Квесты

- kill / boss / power / rebirth
- Награды: coins, power, меч, слот пета, unlock локации

---

## 10. Подземелья

- Easy / Medium / Hard
- Таймер врат
- Coins, power, relic, milestone → pet slot

---

## 11. Локации

- 1 = Тёмный лес (полная)
- 2+ unlock по силе / квесту
- Свои мобы, кейсы, сложность
